# RescueRoute – Real-Time Emergency Response Platform

## Overview
RescueRoute is a full-stack emergency response platform that enables real-time ambulance tracking, hospital coordination, emergency alerts, and route optimization to improve emergency response efficiency.

## Features
- Real-time ambulance tracking
- Hospital coordination
- Emergency alert system
- AI chatbot assistance
- Traffic-aware route optimization
- Responsive user interface

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Real-Time Communication
- Socket.IO
